multiJoin(*otherCurves, blend: bool = False, keepMultipleKnots=True)¶

Runs chained :method:`join` operations. No auto-reversals are performed; input curves must have the correct direction already.

Note

This does not run all the curves through a single attachCurve node, as I’ve run into problems using that in the past.

Parameters:

  • otherCurves – the curves to tack-on; if shapes are passed, the local output is used

  • blend/bl – use the ‘blend’ method rather than ‘connect’; defaults to False

  • keepMultipleKnots/kmk – keep multiple knots; defaults to True

Returns:

The combination curve.
