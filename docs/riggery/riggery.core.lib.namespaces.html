riggery.core.lib.namespaces module¶

Tools to manage Maya namespaces.

exception riggery.core.lib.namespaces.InternalNamespaceError¶

Bases: NamespaceError

The specified namespace can’t be deleted or created because it’s one of the Maya ‘internal’ namespaces, namely ‘:UI’ or ‘:shared’.

class riggery.core.lib.namespaces.Namespace(namespace)¶

Bases: str

String-like namespace manager.

Notes¶

  • Namespaces are conformed to ‘absolute’ form on instantiation

  • The context manager will always activate relative names on enter, and restore the setting on exit.

    If an attempt is made to enter a non-existent namespace, the namespace will be automatically created (‘actualized’). If you need more control over its creation, enter the output of create() instead.

actualize()¶

If this namespace doesn’t exist, creates it. :return: self

addNodes(*nodes)¶
Parameters:

*nodes – the nodes to add to this namespace, packed or unpacked

Rtype *nodes:

Elem or str

classmethod create(namespace: str, makeUnique: bool = False, reuse: bool = False, reuseOnlyIfEmpty: bool = False)¶
Parameters:

  • namespace – the namespace to create

  • makeUnique/mu – if the namespace already exists, append a number to it to make it unique; this will only be used

  • reuse/re – if the namespace already exists, return the existing namespace; defaults to False

  • reuseOnlyIfEmpty/roe – ignored if reuse if False; if the namespace already exists, use it only if it doesn’t contain any nodes (or other namespaces with nodes); otherwise, if makeUnique is True, create a unique name; otherwise, error; defaults to False

Raises:

NamespaceExistsError – The namespace already exists.

destroy(members: bool = False)¶

Removes this namespace; content will be moved to the root namespace; some nodes may be renamed in case of clashes. If this is the current namespace, the current namespace will be switched to the root.

Parameters:

members – delete the namespace members too; defaults to False

Raises:

NamespaceError – The root namespace can’t be deleted.

exists() → bool¶
Returns:

True if this namespace exists.

findNode(pattern: str, recurse: bool = False) → Elem | None¶
Parameters:

  • pattern – a glob-style, local pattern

  • r (recurse /) – look inside child namespaces too; defaults to False

Returns:

The first node under this namespace that matches pattern.

findNodes(pattern: str, recurse: bool = False) → Generator[Elem, None, None]¶
Yields Elem instances for any nodes in this namespace that match

the specified pattern.

Parameters:

  • r (recurse /) – look inside child namespaces too; defaults to False

  • pattern – a glob-style, local pattern

classmethod getAll(includeInternal: bool = False)¶
Parameters:

includeInternal – include the internal, non-user-editable ‘:UI’ and ‘:shared’ namespaces; defaults to False

Returns:

Namespaces excluding the root (‘:’) namespace.

getChildNamespaces(recurse: bool = False, includeInternal: bool = False)¶
Parameters:

  • r (recurse /) – list children of children as well; defaults to False

  • ii (includeInternal /) – include reserved Maya namespaces such as ‘:UI’ and ‘:shared’; defaults to False

Returns:

Any namespaces under this one.

classmethod getCurrent() → Namespace¶

Class method. Returns the current namespace.

getMembers(nodes: bool | None = None, namespaces: bool | None = None, recurse: bool = False)¶

Internal Maya nodes (e.g. item filters) are not included.

The nodes and namespaces flags are evaluated Maya-style, i.e. if only one is passed in as True, the other is set to False.

Parameters:

  • nodes – include nodes

  • r (recurse /) – look inside nested namespaces too

  • namespaces – include nested namespaces

getNodes(recurse: bool = False)¶
Parameters:

r (recurse /) – look inside child namespaces too; defaults to False

Returns:

A list of nodes under this namespace.

Returns:

This namespace’s immediate parent. If this is the root namespace, None.

getParents()¶

List version of iterParents().

hasParent(parentNamespace: str | Namespace) → bool¶
Returns:

True if parentNamespace is a parent to this namespace, regardless of rank.

isCurrent() → bool¶
Returns:

True if this is the current namespace.

isEmpty() → bool¶

True if this namespace is empty of nodes or child namespaces.

isInternal() → bool¶
Returns:

True if this is one of Maya’s reserved ‘internal’ namespaces, namely ‘:UI’ or ‘:shared’.

isRoot() → bool¶
Returns:

True if this is the root namespace.

iterParents() → Generator[Namespace, None, None]¶

Iterates over this namespace’s parents, up to and including the root namespace. If this is the root namespace, nothing will be yielded.

classmethod pruneNamespaces()¶

Removes any namespaces which don’t contain nodes or other non-empty namespaces.

Creates a new namespace, moves all of this namespace’s members to that, deletes this namespace, and returns the new namespace.

class riggery.core.lib.namespaces.RelativeNames¶

Bases: object

Context manager; turns on ‘relative names’ for the block and restores the previous setting on exit.

Note that Namespace also does this by default, so no need to combine them.

riggery.core.lib.namespaces.abs(namespace: str) → str¶

Doesn’t perform any checks. If the namespace doesn’t start with ‘:’, it will have the current namespace prepended to it.

riggery.core.lib.namespaces.clean(namespace: str) → str¶
