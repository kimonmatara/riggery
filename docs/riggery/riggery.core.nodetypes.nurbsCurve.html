classmethod create(points: Iterable, degree=None, parent=None, periodic: bool = False, name: str | None = None, worldSpace: bool = False, ep: bool = False, displayType=None, lineWidth: float | None = None)¶
Parameters:

  • points – the CV points; these can be plugs (for a ‘live’ curve) or values

  • degree/d – if omitted, defaults to 3, but drops to 2 or 1 when there aren’t enough CVs

  • parent/p – an optional parent for the curve shape; defaults to None

  • periodic/per – whether to create a fully closed (periodic) curve; defaults to False

  • name/n – if provided, and a parent is provided, will be used as the shape name; if a parent is not provided, it will be used to name the newly-generated parent; if omitted, block naming will be used, where available; defaults to False

  • displayType/dt – sets the override display type; defaults to None

  • lineWidth/lw – an optional value for the display line width; defaults to None

Raises:

ValueError – Fewer than two CVs were requested, or the number of CVs is not possible given the requested degree.

Returns:

The generated curve shape.
