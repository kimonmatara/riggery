addTripleAttr(basename: str, *, suffixes: str | Iterable[str] = 'XYZ', keyable: bool = False, channelBox: bool = False, input=None, defaultValue=None, asAngle: bool = False, asDistance: bool = False, section: str | None = None, lock: bool = False, multi=False)¶

Creates a triple compound that mimics the standard translate / rotate / scale attributes on transform nodes.

Parameters:

  • name – a name for the new attribute

  • keyable/k – make the attribute keyable; defaults to False

  • channelBox – make the attribute settable; defaults to False

  • input/i – an optional input; defaults to None

  • defaultValue/dv – an optional default value; defaults to None

  • lock/l – lock the attribute after creation; defaults to False

  • asAngle – use doubleAngle for the child types; defaults to False

  • section/s – the name of the section under which to collect this attribute; if the section doesn’t exist, it will be created

  • asDistance – use doubleLinear for the child types; defaults to False

Returns:

The compound attribute.
