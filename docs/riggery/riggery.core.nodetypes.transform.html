attr(attrName: str, *, checkShape: bool = True)¶

For parity with PyMEL, this does not auto-expand to element 0 on ‘multi’ roots.

Parameters:

  • attrName – the short or long name of the attribute

  • checkShape – look for the attribute on the first shape under this transform too; defaults to True

classmethod create(*, name: str | None = None, matrix=None, parent=None, rotateOrder=None, displayLocalAxis: bool = False, displayHandle: bool = False, zeroChannels: bool = False, worldSpace: bool = False)¶
Parameters:

  • name/n – if omitted, defaults to Name blocks

  • matrix/m – the initial node matrix; defaults to identity

  • parent/p – a destination parent; defaults to None

  • rotateOrder/ro – the initial rotate order; defaults to ‘xyz’

  • displayLocalAxis/dla – display the local transformation axes; defaults to False

  • displayHandle/dh – displays a basic black crosshair; defaults to False

  • zeroChannels/zc – don’t apply matrix to SRT channels, but rather to ‘offsetParentMatrix’; defaults to False

  • worldSpace/ws – apply matrix in world-space; defaults to False

Returns:

The transform node.

setControlShape(libraryKey: str, *, shapeScale=None, shapeColor: None | int | str = None, shapeAxisRemap=None)¶
Parameters:

  • libraryKey – the name of the shape library entry

  • shapeScale/ss – an optional scaling factor for the control shape; defaults to 1.0

  • shapeColor/sc – an optional color index for the control shape; if string, must be one of the keys returned by getControlColorKeys(); defaults to None

  • shapeAxisRemap/sar – if provided, should be two or four letter axes, defining pairwise remaps for the control shape; defaults to None

Returns:

The generated shape(s).

setMatrix(matrix, *, translate=None, rotate=None, scale=None, shear=None, zeroChannels: bool = False, preserveChannels: bool = False, worldSpace: bool = False)¶
Parameters:

  • translate/t – edit translation (by omission)

  • rotate/r – edit rotation (by omission)

  • scale/s – edit scale (by omission)

  • shear/sh – edit shear (by omission)

  • worldSpace/ws – apply matrix in world space; defaults to False

  • zeroChannels/zc – edit ‘offsetParentMatrix’, and zero-out the SRT channels; defaults to False

  • preserveChannels/pc – edit ‘offsetParentMatrix’, and keep SRT channels at current values; defaults to False

Returns:

self
