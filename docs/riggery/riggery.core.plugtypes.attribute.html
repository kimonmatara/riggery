Returns the value of this plug.

Warning

In riggery, value setting / getting defaults to native units.

Parameters:

  • frame – return the value at the specified frame; defaults to the current frame

  • unit – the unit to use; defaults to native units, unless ui is passed as True

  • ui – return the value in UI units; defaults to False

  • rotateOrder/ro – for euler rotation attributes, an override for the embedded rotate order (this is not a reordering operation); if the attribute is the rotate channel on a transform node, defaults to the node’s rotate order; otherwise, defaults to 0 (‘xyz’)

  • asString – for enums, return the enum label rather than the integer value; defaults to False
