riggery.core.datatypes.vector module¶

class riggery.core.datatypes.vector.Vector(arg=None, **kwargs)¶

Bases: Tensor3

angleTo(otherVector, normal=None, *, shortest=False)¶
Parameters:

  • otherVector – the vector towards which to measure an angle

  • normal – if this is provided then, if shortest is True, the angle will be in the -180 -> +180 range; otherwise, it will be in the 0 -> 360 range; if omitted, it will be in the 0 -> 180 range; defaults to None

  • shortest – ignored if normal is omitted

asTranslateMatrix()¶
Returns:

A matrix with the w (position) row set to this vector / point.

blend(other, weight=0.5, slerp: bool = False, preserveLength: bool = False, blendLength: bool = False)¶

Blends this vector towards other.

Parameters:

  • other – the vector towards which to blend

  • weight – the blending weight towards other

  • slerp – perform quaternion-based slerping; defaults to False

  • blendLength – blend the vector lengths as well; defaults to False

  • preserveLength – preserve this vector’s length; defaults to False

Returns:

cross(other, normalize: bool = False)¶
Parameters:

other – the other vector

Returns:

The cross product of self and other.

dot(other, normalize: bool = False)¶
Parameters:

  • other – the other vector

  • normalize – normalize inputs; you’ll usually want this to be True, but defaults to False for parity with the API

Returns:

The cross product of self and other.

flipIfCloserTo(refVector)¶
Parameters:

refVector – the vector to compare to

Returns:

Either self, or the inverse, if the inverse is more closely-aligned to refVector.

guessUpVector()¶

Runs comparisons against base X, Y and Z and vectors, and returns the one that’s most perpendicular to this vector.

length()¶
Returns:

The length of this vector.

loc(name=None, *, inheritsTransform: bool = True)¶
Parameters:

  • name/n – defaults to name blocks

  • inheritsTransform/it – sets the ‘inheritsTransform’ attribute on the locator; defaults to True

Returns:

A locator (transform) with its translate set to this vector or point.

mostPerpendicular(others)¶

Returns the vector output that’s most perpendicular to this one. At the moment this doesn’t support mixed-mode (values mixed with plugs).

normal()¶
Returns:

A normalized copy of this vector or point.

The projection of this vector onto otherVector.

rejectFrom(other, preserveLength=False)¶

The rotated vector

transport(startTangent, endTangent, perpendicularize: bool = True)¶

Performs single-step parallel transport.

Parameters:

  • startTangent – the starting tangent

  • endTangent – the tangent onto which to transport the vector

  • perpendicularize/per – pass False only if you know that this vector is already perpendicular to startTangent; defaults to True

Returns:

This vector, transported onto endTangent.
